Text to speech

GUI is created by using Tkinder python module

Developed a simple text-to-speech (TTS) application in Python using the pyttsx3 library.

The program prompts the user to input text for speech synthesis and utilizes the pyttsx3 library to convert the text into spoken words. The user can enter multiple lines of text until they type 'QUIT' to exit the program.

The project demonstrates proficiency in working with external libraries, user input handling, and basic text-to-speech functionality.
